# Data Inspection and Assumptions  
1. Successfully connected the Excel dataset to Tableau.  
2. Verified that order_date and ship_date are recognized as Date fields.  
3. Geographic fields (region, state, city) were used for geographic analysis.  
4. Numerical measures include Sales, Profit, Quantity, Discount, Delivery Days, and Customer Rating.  
5. return_flag is treated as a binary field where:  
     0 = Order Not Returned  
     1 = Order Returned  
6. Each order_id represents a unique customer order.  
7. Sales and Profit values are assumed to be in the same currency.  
8. Delivery Days represent the number of days between the order date and ship date.
